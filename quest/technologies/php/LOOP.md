f(0) = 0 円
f(1) = 1 円
f(n) = f(n-1) + f(n-2) 円   (n ≧ 2)

```php
n：整数

1 <= n <= 30
```

▼アウトプット

```php
fibonacci(0);
fibonacci(1);
fibonacci(2);
fibonacci(3);
fibonacci(4);
fibonacci(7);
fibonacci(30);
```

▼サンプルアウトプット

```php
0
1
1
2
3
13
832040
```

▼回答例

```php

{

}
```
