# 変数を宣言し代入することができる

## 1. 時間

秒単位の時間 s を、h:m:s の形式へ変換して出力してください。ここで、h は時間、m は 60 未満の分、s は 60 未満の秒とします。

▼インプット

```php
s：整数

0 ≤ s < 86400
```

▼アウトプット

h、m、s を : 区切りで1行に出力してください。

▼サンプル呼び出し

```php
time(4210);
```

▼サンプルアウトプット

```php
1:10:10
```

▼回答例

```php
function time(s)
{
    # 関数を完成させてください
}
```
