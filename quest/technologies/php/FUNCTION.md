# 関数を自作し使うことができる

## 1. Hello World

"Hello World" と標準出力（画面）に出力する関数 hello を定義してください。

▼インプット

入力はありません。

▼アウトプット

"Hello World" と出力してください。

▼サンプル呼び出し

```php
hello();
```

▼サンプルアウトプット

```php
Hello World
```

▼回答例

```php
function hello()
{
    # 関数を完成させてください
}
```

## 2. 挨拶

名前を引数に受け取り、与えられた名前の挨拶メッセージを標準出力（画面）に出力する greeting という関数を書きなさい。

▼インプット

```php
name：文字列
```

▼アウトプット

"おはよう、◯◯！" と出力してください。

▼サンプル呼び出し

```php
greeting('渡辺');
```

▼サンプルアウトプット

```php
おはよう、渡辺！
```

▼回答例

```php
function greeting(name)
{
    # 関数を完成させてください
}
```
