# 変数を宣言し代入することができる

## 1. 時間

秒単位の時間 s を、h:m:s の形式へ変換して出力してください。ここで、h は時間、m は 60 未満の分、s は 60 未満の秒とします。

▼インプット

```ruby
s：整数

0 ≤ s < 86400
```

▼アウトプット

h、m、s を : 区切りで1行に出力してください。

▼サンプル呼び出し

```ruby
time(4210)
```

▼サンプルアウトプット

```ruby
1:10:10
```

▼回答例

```ruby
def time(s)
  # 関数を完成させてください
end``
```
