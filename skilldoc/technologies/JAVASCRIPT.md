# JavaScript

JavaScript の習得を目指します。

JavaScript は Web サービスのフロントエンドを作成する際に必須の技術です。

JavaScript の技術領域は非常に広範にわたります。JavaScript の基本文法と DOM 操作を身につけた上で、非同期処理やフレームワーク、TypeScript などを扱い、モダンな JavaScript の開発に必要なスキルを身につけていきましょう。
